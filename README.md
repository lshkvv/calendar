# Личный календарь (Qt + SQLite)

Небольшое десктоп‑приложение на Qt для управления личными событиями, дедлайнами и тегами.

## Стек

- C++17
- Qt 6 (Widgets, Sql)
- SQLite (через QtSql)
- CMake

## Возможности

- Категории событий с настраиваемым цветом
- Список событий с категорией и тегами
- Множество тегов на одно событие (связь многие‑ко‑многим)
- Цветные плашки категорий в списке событий

## Сборка

```bash
mkdir build && cd build
cmake ..
cmake --build .
./calendar
```

Qt должен быть установлен и доступен для CMake (`Qt6_DIR` в `CMAKE_PREFIX_PATH` или через Qt Creator).
